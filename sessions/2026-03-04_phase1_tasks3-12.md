## Architecture Notes

### `core/config.py` + `tests/conftest.py` pattern
Config raises `ValueError` at import time if required env vars are missing. `tests/conftest.py` sets fake values at **module level** (before any test file imports) using `os.environ.setdefault()`. This allows all unit tests to import config without real API keys, while real keys from shell env take precedence.

### `audioop-lts` for Python 3.13
`pydub` depends on `audioop` which was removed from Python 3.13 stdlib. `audioop-lts` is the maintained backport. Added to `requirements.txt`.

### Integration test design
`tests/integration/test_pipeline.py` patches all 5 external API layers:
1. `arxiv.Client` → returns fixture paper objects
2. `SemanticScholar` → returns mock S2 data
3. `anthropic.Anthropic` → 3 separate mocks (bibliography, graph, script agents each import their own client)
4. `ElevenLabs` → returns MP3 bytes via `side_effect` lambda
5. `DATA_DIR` → redirected to `tmp_path` so tests don't pollute `data/`

Also patches `GRAPH_PATH` and `GRAPH_SNAPSHOT_PATH` in `core.knowledge_graph` module so graph files go to tmp.
